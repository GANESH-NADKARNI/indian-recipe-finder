# 🍛 Indian Recipe Finder

An interactive web application that helps users discover authentic Indian recipes based on ingredients and cuisine preferences.

Users can search recipes by ingredient, filter by cuisine type, and view detailed cooking instructions through an intuitive interface.

---

## ✨ Features

* Search recipes by ingredient
* Filter recipes by cuisine
* Responsive recipe cards
* Detailed recipe popup modal
* Clean and modern UI
* Fast client-side filtering
* Beginner-friendly implementation

---

## 📂 Project Structure

```text
Indian-Recipe-Finder/
├── index.html
├── styles.css
└── script.js
```

---

## 🍽️ Supported Categories

* North Indian
* South Indian
* Punjabi
* Hyderabadi
* Kashmiri
* Chaat
* Dessert

---

## 🔍 Search Functionality

Users can search recipes using ingredients such as:

```text
paneer
dal
rice
chicken
potato
tomato
```

The application instantly filters matching recipes.

---

## 📖 Recipe Details

Each recipe includes:

* Recipe name
* Cuisine type
* Ingredients
* Cooking instructions
* Additional recipe information

Displayed through an interactive modal window.

---

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/GANESH-NADKARNI/indian-recipe-finder.git
```

### Run the Application

Open:

```text
index.html
```

in any modern web browser.

No installation or backend setup is required.
